# 🐍 Snake Game

A classic Snake game built with vanilla HTML, CSS, and JavaScript — no libraries, no frameworks, just clean browser code.

## Preview

Dark grid background with a glowing green snake and a pulsing red apple. Score tracked live in the title.

## Features

- Smooth grid-based movement
- Glowing snake with gradient fade toward the tail
- Pulsing red food with glow effect
- Dark themed board with subtle grid lines
- Live score display
- Game over alert with final score

## How to Play

1. Clone or download the repo
2. Open `index.html` in your browser
3. Press any arrow key to start moving
4. Eat the red food to grow and score points
5. Avoid hitting the walls or yourself

### Controls

| Key | Action |
|-----|--------|
| ↑ Arrow | Move up |
| ↓ Arrow | Move down |
| ← Arrow | Move left |
| → Arrow | Move right |
